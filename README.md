# 📊 E-Commerce Sales Analysis Dashboard

## 🚀 Project Overview
This project focuses on analyzing e-commerce sales performance, customer behavior, and retention trends using interactive Power BI dashboards.

The dashboard provides valuable business insights through sales analysis, customer retention analysis, and cohort-based performance tracking.

---

# 🎯 Objectives
- Analyze overall sales and profit performance
- Track monthly revenue trends
- Identify top-performing categories and customers
- Understand customer retention behavior
- Perform cohort analysis for retention tracking
- Support business decision-making using data visualization

---

# 🛠️ Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel / CSV
- Cohort Analysis
- RFM Analysis

# 💡 Key Insights

- 📈 Total revenue reached **432K** with **24K** total profit.
- 🏆 Electronics category generated the highest revenue contribution.
- 👥 Repeat customers contributed nearly **68%** of the customer base.
- 📉 Customer retention decreases gradually after initial purchases.
- 📊 Revenue showed strong growth during late 2018 and early 2019.
- 💰 A small group of customers contributed significantly to overall revenue.
